Ext.iterate надстройка над Ext.each, позволяет делать цикл по массиву или объекту, сама определяет, что вызывать Ext.Array.each или Ext.Object.each

Имеется массив объектов [{ x: ‘val 1’, y: 2 }, { x: ‘val 5’, y: 5 }, { x: ‘val 2’, y: 4 }]. Каким образом получить массив из значений поля x средствами ExtJS?

var testArr = [{ x: 'val 1', y: 2 }, { x: 'val 5', y: 5 }, { x: 'val 2', y: 4 }]; var result = Ext.Array.map(testArr, function(item){ return item.x; });

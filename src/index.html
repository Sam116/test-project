Даны две переменные a и b. Например, они равны a=5; b=3. Необходимо поменять местами эти числа, но не используя третью переменную.

a=5; b=3; b = [a, a = b][0]; // Или так [a, b] = [b, a]; // Или так a = a + b - (b = a);

7.

Имеется массив var arr = [1, 2, 4, 5]. Посчитайте сумму элементов массива.

var arr = [1, 2, 4, 5]; var result = arr.reduce(function(sum, current) { return sum + current; }, 0);

Имеется массив var arr = [1, 2, 3, 1, 3, 4]. Отфильтруйте только по уникальным значениям.

var arr = [1, 2, 3, 1, 3, 4]; // Es6 var result = Array.from(new Set(arr)); // Или так var result = arr.filter(function(item, index){ return arr.indexOf(item) === index });
